# Architecture Decision Records

The current project decisions are recorded in `PROJECT_CHARTER.md` as `ADR-PMA-*` entries.

When implementation begins, new or materially revised decisions should be added as standalone records in this directory using immutable identifiers and this minimum structure:

```text
ID
Status
Date
Context
Decision
Alternatives considered
Consequences
Supersedes / Superseded by
Related contract, schema, fixture, or pull request
```

Do not rewrite accepted historical decisions to make later outcomes appear inevitable. Supersede them with a new record when the decision changes.
